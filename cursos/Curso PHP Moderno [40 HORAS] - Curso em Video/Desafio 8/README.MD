# Desafio 08 – Cálculo de Raiz Quadrada e Raiz Cúbica em PHP

## Descrição

Este desafio integra o Curso de PHP Moderno, Módulo 01, ministrado por Gustavo Guanabara, na plataforma Curso em Vídeo. O objetivo é desenvolver uma aplicação capaz de calcular a raiz quadrada e a raiz cúbica de um número informado pelo usuário, utilizando conceitos matemáticos aplicados à programação.

O exercício reforça o entendimento sobre potenciação e demonstra como implementar cálculos matemáticos no PHP, incluindo situações em que não existe uma função nativa específica, como no caso da raiz cúbica.

---

## Objetivo

Permitir que o usuário informe um número real e receba como resultado:

O valor da raiz quadrada do número informado  
O valor da raiz cúbica do número informado  

Apresentando os resultados com formatação numérica adequada.

## Funcionamento da Aplicação

O funcionamento da aplicação segue o fluxo abaixo:

O usuário informa um número por meio de um formulário HTML  
O formulário é retroalimentado para manter o valor informado após o envio  
O PHP captura o valor utilizando o método GET  
A raiz quadrada é calculada utilizando a função sqrt  
A raiz cúbica é calculada utilizando o operador de potência com expoente fracionário  
Os resultados são formatados utilizando number_format para melhor visualização  
Os valores calculados são exibidos diretamente ao usuário

## Aprendizados Importantes

A raiz quadrada pode ser calculada utilizando função nativa ou potenciação  
A raiz cúbica pode ser obtida elevando o número ao expoente fracionário correspondente  
Compreender os conceitos matemáticos facilita a implementação de soluções programáveis  
A formatação correta dos números melhora a apresentação e interpretação dos resultados  
Testar cálculos matemáticos passo a passo reduz erros e aumenta a confiabilidade do código
