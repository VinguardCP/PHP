# Desafio 01 – Antecessor e Sucessor

Este desafio consiste em desenvolver um pequeno sistema em **PHP** que recebe um número digitado pelo usuário e exibe o seu **antecessor** e **sucessor**.

Apesar da simplicidade aparente, o desafio é fundamental para a compreensão de conceitos básicos da linguagem e da integração entre **HTML, CSS e PHP**.

---

## Objetivo do Desafio

- Receber um número informado pelo usuário
- Processar o valor no servidor utilizando PHP
- Exibir o número digitado, seu antecessor e seu sucessor

---

## Tecnologias Utilizadas

- PHP
- HTML
- CSS
- JavaScript (básico)
- Servidor local (XAMPP)
